1. Tạo repository GitHub
Sinh viên đã tạo một repository trên GitHub nhằm lưu trữ toàn bộ bài tập thực hành của môn học.

File README.md được dùng để ghi nhận và báo cáo kết quả thực hành của từng tuần.
Link repository: Tuan-1 Repository

Khả năng phân biệt nội dung
Chất lượng hiển thị giao diện người dùng (UI)

Đạt số điểm cao nhất có thể trong các bài test đánh giá UI/UX.
